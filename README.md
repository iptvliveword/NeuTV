# NEU TV

使用校园网便捷的观看网络电视 Android 移动版

[直视tv 下载链接 v2.2](https://github.com/xfangfang/NeuTV/releases/download/2.2/app-release.apk)


### 简介

![shot1](demo1.jpeg)
![shot2](demo2.jpeg)


直视是一款集成电视直播与回看的Android应用，拥有长达七天的直播视频回看期限，在拥有IPV6校园网络的各个地方都能便捷的观看电视节目。

### 功能

- 频道收藏功能，好看的频道再也不要找不见
- 重绘应用界面，一定要美美美～
- 北影大佬“老头子”制作全新应用图标
- 节目单分类，快读定位您想看的节目
- 播放中切换频道，提升用户体验
- 节目源不好？换！
- 更多功能，敬请期待
